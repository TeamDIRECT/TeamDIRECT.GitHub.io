0 - Assicurarsi di aver installato Google Chrome
1 - Chiudere ogni istanza di Chrome o di altro browser aperta
2 - Aprire il file launcher.ink si aprirà una finestra di Chrome
3 - Trascinare con un drag and drop il file index.html dentro la finestra aperta precedentemente
4 - Attendere qualche secondo per il caricamento del modello

NB: il file launcher.ink non è altro che un collegamento all'eseguibile di Chrome, con permessi per visualizzazione file in locale, ciò è ottenuto aggiungendo la stringa --allow-file-access-from-files nel campo "Destinazione" della scheda "Collegamento" delle proprietà del file (Vedere screenshoot allegato)
